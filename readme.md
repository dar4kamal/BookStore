[APi Docs](https://github.com/dar4kamal/BookStore/blob/master/Api_Docs/API_DOCS.md)

## Start App
### Install npm dependencies
```
npm install
```
### Create .env File in root directory
```
PORT=5000
MONGO_URL='mongodb://localhost:27017'
MONGO_DB_NAME = 'BookStore'
```
